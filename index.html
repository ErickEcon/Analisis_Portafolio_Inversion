En esta etapa se importaron los datos de la fuente de Yahoo Finance. Se han seleccionado los siguientes activos para el periodo de anlisis (2020-2025):

  • SPY: S&P 500 que se utiliza como Benchmark y proxy de Renta Vaiable
  • TLT: Bonos del Tesoro 20+ aos que acta como Refugio y es proxy de Renta Fija
  • QQQ: Nasdaq 100 como un activo de alto crecimiento en los ltimos aos, perteneciente al sector de Tecnologa

Para este tipo de anlisis es importante considerar un benchmark para medir y comparar el rendimiento de los portafolios y, en la literatura actual, el ETF SPY es el ms utilizado. Por otra parte, es usual y til incluir activos de renta fija, como los bonos del Tesoro de 20 aos, como activo de refugio porque histricamente se ha demostrado que existe una correlacin negativa con los activos de renta variable. Por ltimo, se ha decidido incluir el Nasdaq 100, el cual representa principalmente el sector de Tecnologa y este ha tenido un rpido crecimiento en los ltimos aos. En resmen, la construccin de este portafolio cuenta con un componente principal que es el SPY, un activo defensivo el TLT y un activo de crecimiento el QQQ para analizar cmo la diversificacin afecta el ratio Sharpe.

# Asumimos 252 das de trading al ao
dias_trading = 252
risk_free_rate = 0.04  # 4% anual aprox (T-Bills actuales)

# 1. Crear un DataFrame vaco para guardar las mtricas
metrics = pd.DataFrame(index=tickers, columns=['Retorno Total', 'CAGR (Anual)', 'Volatilidad', 'Sharpe Ratio', 'Max Drawdown'])

# 2. Clculo del Retorno Total
metrics['Retorno Total'] = (data.iloc[-1] / data.iloc[0]) - 1

# 3. Clculo del CAGR (Compound Annual Growth Rate)
# Nmero de aos en la muestra
years = (data.index[-1] - data.index[0]).days / 365.25
metrics['CAGR (Anual)'] = (1 + metrics['Retorno Total'])**(1/years) - 1

# 4. Clculo de Volatilidad Anualizada
# Desviacin estndar diaria * raz cuadrada de 252
metrics['Volatilidad'] = daily_returns.std() * np.sqrt(dias_trading)

# 5. Clculo del Ratio de Sharpe
# (Retorno Anual - Tasa Libre de Riesgo) / Volatilidad
metrics['Sharpe Ratio'] = (metrics['CAGR (Anual)'] - risk_free_rate) / metrics['Volatilidad']

# 6. Clculo del Mximo Drawdown (La parte ms compleja)
# Calculamos el pico acumulado hasta la fecha
rolling_max = data.cummax()
# Calculamos la cada porcentual desde ese pico
drawdown = (data / rolling_max) - 1
# Nos quedamos con la cada mnima (la ms negativa)
metrics['Max Drawdown'] = drawdown.min()

# --- Formato Visual ---
# Vamos a dar formato de porcentaje para que se vea profesional
pd.options.display.float_format = '{:.2%}'.format

#7. --- CLCULO DEL VALUE AT RISK (VaR) ---

# 1. Configuracin de la Simulacin
num_portafolios = 5000
all_weights = np.zeros((num_portafolios, len(tickers)))
ret_arr = np.zeros(num_portafolios)
vol_arr = np.zeros(num_portafolios)
sharpe_arr = np.zeros(num_portafolios)

# 2. Simulacin de Monte Carlo
print("Simulando 5,000 combinaciones de portafolio...")

np.random.seed(42) #para propsitos de reproducibilidad
for ind in range(num_portafolios):
    # Crear pesos aleatorios
    weights = np.array(np.random.random(len(tickers)))
    # Ajustar para que sumen 100%
    weights = weights / np.sum(weights)
    
    # Guardar pesos
    all_weights[ind, :] = weights
    
    # Calcular Retorno Esperado (Logaritmo)
    # Asumimos que el retorno futuro ser similar al promedio histrico
    ret_arr[ind] = np.sum((daily_returns.mean() * weights) * 252)
    
    # Calcular Volatilidad Esperada (Matriz de Covarianza)
    # Esta frmula es lgebra lineal pura: w^T * Cov * w
    vol_arr[ind] = np.sqrt(np.dot(weights.T, np.dot(daily_returns.cov() * 252, weights)))
    
    # Calcular Sharpe Ratio
    sharpe_arr[ind] = (ret_arr[ind] - risk_free_rate) / vol_arr[ind]

# 3. Encontrar el Portafolio ptimo (El de mayor Sharpe)
max_sharpe_idx = sharpe_arr.argmax()
max_sharpe_return = ret_arr[max_sharpe_idx]
max_sharpe_vol = vol_arr[max_sharpe_idx]
optimal_weights = all_weights[max_sharpe_idx, :]

# 4. Mostrar Resultados Numricos
print("\n--- PORTAFOLIO PTIMO SUGERIDO ---")
print(f"Retorno Esperado: {max_sharpe_return:.2%}")
print(f"Volatilidad:      {max_sharpe_vol:.2%}")
print(f"Sharpe Ratio:     {sharpe_arr[max_sharpe_idx]:.2f}")
print("-" * 30)
print("Asignacin de Activos Ideal:")
for i, ticker in enumerate(tickers):
    print(f"{ticker}: {optimal_weights[i]:.2%}")

# 5. Visualizacin (La Frontera Eficiente)
plt.figure(figsize=(10, 6))
# Scatter plot de todos los portafolios simulados
plt.scatter(vol_arr, ret_arr, c=sharpe_arr, cmap='viridis', alpha=0.5)
plt.colorbar(label='Sharpe Ratio')

# Destacar el punto ptimo con una estrella roja
plt.scatter(max_sharpe_vol, max_sharpe_return, c='red', s=100, marker='*', label='Portafolio ptimo')

plt.title('Frontera Eficiente (Optimizacin de Portafolio)')
plt.xlabel('Volatilidad (Riesgo)')
plt.ylabel('Retorno Esperado')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
